# Umar Abdulmajeed | Portfolio 

A modern, high-performance portfolio website built for a Web Designer & Social Media Manager. Designed with a focus on dark-mode aesthetics, responsive layouts, and user conversion.

**[Live Demo](https://godsgraceultra.github.io/webmar/)**

## Tech Stack
* **HTML5** - Semantic structure
* **Tailwind CSS** - Modern styling and responsiveness
* **FontAwesome** - Professional iconography
* **JavaScript** - Interactive mobile navigation

## Features
* **Fully Responsive:** Optimized for mobile, tablet, and desktop.
* **Modern UI:** Glassmorphism navigation and gradient-aura image effects.
* **Optimized UX:** Smooth scroll navigation and clear Call-to-Action (CTA) paths.
